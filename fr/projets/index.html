mlscores

GitHub

mlscores genere des scores de multilinguisme pour les elements Wikidata, notamment pour les proprietes et les libelles des valeurs de proprietes. Les scores representent le pourcentage de libelles disponibles dans une langue donnee.

GitHub

ShExStatements permet de generer des expressions de forme a partir d'enonces CSV simples, de fichiers CSV et de feuilles de calcul. Il peut etre utilise en ligne de commande ainsi que via une interface web.

WDProp

GitHub

WDProp fournit une interface visuelle pour comprendre et ameliorer le developpement d'ontologies multilingues et collaboratives sur Wikidata. Son objectif est d'analyser differents aspects des proprietes Wikidata, en particulier le processus de traduction.
