Jupyter

Installation

Exécutez les commandes suivantes sur votre machine virtuelle ou sur un PC. L'objectif est d'installer pip qui facilite l'installation les autres outils et les bibliothèques Python.

pip

$ sudo apt install python-pip

jupyter

Nous utilisons les notebooks jupyter pendant toutes les séances de TP. Pour l'installer, nous utiliserons pip ou python3 comme expliqué par la suite.

$ python3 -m pip install jupyter

ou

$ pip3 install jupyter

Si vous souhaitez installer jupyter que sur votre compte personnel, utilisez l'option --user. Les autres utilisateurs n'auront pas l'accès.

$ pip3 install --user jupyter

Hello World!

Une fois jupyter est installé, vous pouvez le tester en exécutant les commandes suivantes. L'objectif est de créer un dossier TP1 et d'exécuter la commande jupyter dans ce dossier.

$ mkdir TP1 && cd TP1 $ jupyter notebook

Cliquez sur l'onglet 'Running'.

Allez à l'onglet 'Files' et cliquez sur New et choisir Python3 sous Notebook

Un nouvel onglet s'ouvre.

print("Hello World!")

Cliquez sur n'importe quelle cellule et appuyez 'Run'.

Par défaut, votre Notebook est nommé 'Untitled'. Vous pouvez le renommer en cliquant sur le texte 'Untitled' et en donnant un nouveau nom.

Maintenant allez à l'onglet 'Files' tab et vous pouvez voir votre Notebook.

Jupyterlab

Installation

Récemment, jupyterlab est sorti avec une meilleure interface et avec la possibilité de travailler en mode collaboratif. Comme pour l'installation de jupyter, nous utiliserons pip pour son installation.

$ python3 -m pip install jupyter-lab

ou

$ pip3 install jupyterlab

ou

$ pip3 install --user jupyterlab

Hello World!

$ mkdir TP1 && cd TP1 $ jupyterlab

Cliquez sur 'Python 3' sous 'Notebook' et écrivez le code "Hello World".

Pandas

Installation

Afin de faire l'analyse des données, nous installerons prochainement pandas qui est une bibliothèque très poussée pour faire l'analyse de données structurées.

$ python3 -m pip install pandas

ou

$ pip3 install pandas

ou

$ pip3 install --user pandas

Scikit-learn

Installation

Enfin, nous installerons la bibliothèque Scikit-learn pour faire des analyses avancées. Suivez les commandes suivantes.

$ python3 -m pip install sklearn

ou

$ pip3 install sklearn

ou

$ pip3 install --user sklearn
