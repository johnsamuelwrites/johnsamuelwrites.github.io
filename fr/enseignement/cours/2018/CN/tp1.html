Exercice 1.1 ★

Les exemples suivantes présent les fondamentaux de la programmation en Python.

1. Commentaires

En Python, si vous voulez écrire un commentaire, commencez votre phrase avec un dièse (#).

# Ceci est un commentaire sur une seule ligne

Si vous votre commentaire ne tient pas sur une seule ligne, vous pouvez écrivez votre commentaire en le commencant avec trois guillemets (""") et en le finissant pareil comme indiqué ci-dessous.

"""
  Ceci est un commentaire
  sur plusieurs lignes
"""

Maintenant vous êtes prêt pour votre première bout de code, afin d'afficher "Bonjour" sur l'écran.

print("Bonjour")

2. Variables

Python vous également permet votre message dans une variable.

# une variable
message = "le monde!"
print(message)

L'avantage d'utiliser une variable est la possibilé d'utiliser une valeur plusieurs fois sans le réécrire.

# une variable
message = "le monde!"
print(message)
print(message)

Vous pouvez également changer la valeur d'une variable.

# une variable
message = "le monde!"
print(message)
message = "Bonjour!"
print(message)

Il est possible de stocker différents types de donnés. Prochainement, nous testons des variables en stockant les entiers, afin de les utiliser pour un calcul mathématique.

a = 10
b = 20
c = a + b
print(c)

Le prochaine code stocke un nombre réel dans une variable.

#nombres réels
pi = 3.14
print(pi)

Prochinement, nous testerons les calculs en utilisant les entiers et les nombres réels.

#nombres réels
pi = 3.14
rayon = 5
superficie = pi * rayon * rayon
print(superficie)
perimetre = 2 * pi * rayon
print(perimetre)

Testez le code au-dessus en changeant la valeur de rayon.

Parfois, il est important de tester le type d'une variable. En Python, vous pouvez utiliser une fonction intégrée, type en passant la variable comme argument. Exécutez le code suivant et notez bien les résultats pour les différentes variables.

#type de données
message1 = "Bonjour"
a = 12
pi = 3.14
print(type(message1))
print(type(a))
print(type(pi))

3. Concatenation de chaine de caractères

Il est possible de concatener deux chaînes de caractères en utilisant l'opérateur '+'.

#concatenation de chaine de caractères
message = "le monde!"
print("Bonjour" + message)

Ci-dessous, vous voyez le code que concatene deux schaînes de caractères en utilisant l'opérateur '+' et les variables.

#concatenation de chaine de caractères
message1 = "Bonjour "
message2 = "le monde!"
print(message1 + message2)

Testez le code suivant.

#concatenation de deux différents types de données
message1 = "Bonjour en Python"
a = 3
print(message1 + a)

Quelle est l'erreur que vous avez eue?

Il est impossible de concatener une chaîne de caractères avec un autre type de données (comme, un entier dans l'exemple au-dessus). Python nous fournit une fonction intégrée appelée (str) que peut convertir un entier, un nombre réel à une chaîne de caractères.

#concatenation de deux différents types de données
message1 = "Bonjour en Python "
a = 3
print(message1 + str(a))

4. Liste

Notre prochaine objectif est de voir comment nous pouvons stocker une ou plusieurs valeurs du même type dans une variable. Une liste nous permet de faire ça. Pour créer une liste, mettez toutes les valeurs du même type de données entre '[' ']' et en utilisant les virgules comme séparateur comme indiqué ci-dessous.

a = [10, 20, 30, 40, 50]
print(a)

L'ordre des élements dans une liste est importante. Pour accèder un membres de la liste, nous utiliserons son indice. En Python, l'indice commence à 0; c'est à dire, pour accèder le premièr membre de la liste, utilisez 0. Pour le nième membre, utilisez n-1 etc.

a = [10, 20, 30, 40, 50]
print(a[0])
print(a[1])
print(a[2])
print(a[3])
print(a[4])

Tester le code suivant où on essaie d'accèder un membre inexistant.

a = [10, 20, 30, 40, 50]
print(a[8])

Nous avons vu des chaînes de caractères au-dessus. Mais une chaîne de caractères est également une liste. En conséquence, vous pouvez utilisez les indices pour accèder la nième caractère de la chaîne.

message1 = "Bonjour en Python "
print(message1[0])
print(message1[1])
print(message1[2])
print(message1[3])
print(message1[4])
print(message1[5])
print(message1[6])
print(message1[7])

Utilisez la fonction intégrée len afin d'obtenir la taille d'une chaîne.

message1 = "Bonjour en Python "
print(len(message1))

Vous pouvez utiliser len pour savoir le nombre des membres dans une liste.

a = [10, 20, 30, 40, 50]
print(len(a))

Pour ajouter un nouveau membre dans une liste, utilisez la méthode append. Le nouveau membre est ajouté à la fin de la liste. Testez le code et voyez le résultat.

a = [10, 20, 30, 40, 50]
a.append(60)
print(a)

Il est possible de modifier la nième valeure dans une liste. Précisez le nième membre comme indiqué ci-dessous en utilisant l'indice et affectuez la nouvelle valeur.

a = [10, 20, 30, 40, 50]
a[0] = 0
print(a)

Testez le code suivant pour affectuer une indice inexistante. Quelle est l'erreure que vous avez eue?

a = [10, 20, 30, 40, 50]
a[6] = 20
print(a)

Vous avez vu au-dessus la méthode append qui ajoute le nouveau membre à la fin de la liste. Pourtant avec la méthode insert, nous pouvons insérer un nouveau membre n'importe où dans la liste. Le premier argument d'insert est l'indice et le deuxième c'est la valeure. Testez le code suivant et voyez la nouvelle liste et sa taille après l'insertion.

a = [10, 20, 30, 40, 50]
a.insert(0, 0)
print(a)
print(len(a))

Testez le code suivant avec différents valeurs pour l'indice (premier argument d'insert).

a = [10, 20, 30, 40, 50]
a.insert(6,60)
print(a)
print(len(a))

5. Tuple (une liste qui ne peut plus être modifiée.)

Une liste très intéressant en Python est un tuple. Un tuple est une liste qui ne peut plus être modifiée et déclarer en mettant les membres entre ( et ) au lieu de symboles [ et ] respectivement.

a = (10, 20, 30, 40, 50)
print(a)

Nous pouvons accèder n'importe quel membre d'un tuple en utilisant son indice.

a = (10, 20, 30, 40, 50)
print(a[0])

Essayez changer les valeurs d'un tuple, comme nous avons testé changer une liste au-dessus.

a = (10, 20, 30, 40, 50)
a[0] = 0
print(a)

Quelle est l'erreur que vous avez eue?

Le tuple est très performant à comparer d'une liste. C'est pourquoi, les tuples sont très utilisé pour l'analyse de données. Vous pouvez la méthode pour convertir une liste à un tuple en utilisant la méthode tuple.

a = [10, 20, 30, 40, 50]
b = tuple(a) print(b)
print(type(b))

6. Ensemble (set)

Si vous n'êtes pas interessé par l'ordre des membres dans une liste, mais surtout interéssé par les membres distincts, utilisez une ensemble en Python. Voyez le code suivant où nous mettons plusisurs valeurs différentes, mais nous voyons beaucoup de répitition. Nous utilisons les symboles { et } pour créer une ensemble des valeurs homogènes. Voyez-bien l'affichage de ce code et vous voyez, la variable a contient que des valeurs distinctes.

a = {10, 20, 30, 40, 50, 10, 20, 30, 40, 50}
print(a)

La méthode add est utilisé pour ajouter un nouveau membre dans l'ensemble. Ci-dessous, nous essoyons d'ajouter une valeur déjà existant dans l'ensemble.

a = {10, 20, 30, 40, 50, 10, 20, 30, 40, 50}
a.add(10)
print(a)

Ci-dessous, nous essoyons d'ajouter une valeur inexistant dans l'ensemble. En effet l'ordre n'est pas importante dans une ensemble, la position de nouveau membre est uncertain.

a = {10, 20, 30, 40, 50, 10, 20, 30, 40, 50}
a.add(60)
print(a)

Pour supprimer une valeur d'une ensemble, utilisez la méthode remove.

a = {10, 20, 30, 40, 50, 10, 20, 30, 40, 50}
a.remove(40)
print(a)

Enfin, nous reregardons les différents façons de stocker une collection des valeurs homogènes.

#set
a = {10, 20, 30, 40, 50, 10, 20, 30, 40, 50}
print(a)
print(type(a))

#tuple
b = (10, 20, 30, 40, 50, 10, 20, 30, 40, 50)
print(b)
print(type(b))

#liste
c= [10, 20, 30, 40, 50, 10, 20, 30, 40, 50]
print(c)
print(type(c))

from operator import itemgetter

mots = {"contente": 12, "content": 12, "triste": 2, "joie" : 10}
mots_tuple = [(cle, valeur) for cle,valeur in mots.items()]
print(sorted(mots_tuple, key=itemgetter(1)))
