2. WDProp: Wikidata Property Analysis

  • WDProp
    • Get real-time translation statistics
    • Navigate supported languages, properties, datatypes, classes
    • Compare translation statistics
    • Find available properties for an entity
    • Shows associated Wikidata SPARQL endpoints and Mediawiki API
    • Based on basic web technologies (HTML, Javascript, CSS)
  • URL
    • https://wdprop.toolforge.org/
    • Source code: https://github.com/johnsamuelwrites/wdprop
