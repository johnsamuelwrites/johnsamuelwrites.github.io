Data Science for Chemists

Checklist

Before submitting your project, please ensure that you adhere to the following checklist:

  • ✅ Full names (first and last names) of the team are correctly included in the file CONTRIBUTORS.md.
  • ✅ The README.md file is filled out comprehensively and adheres to the provided instructions.
  • ✅ You have adhered to file names as specified in each lab exercise.
  • ✅ Your code is accompanied by appropriate comments to explain logic and functionality.
  • ✅ Your code can be executed without generating errors, and ideally without generating warnings.
  • ⚠ No Python files (*.py)
