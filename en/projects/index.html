Projects

  • mlscores mlscores generates multilinguality scores for Wikidata items, specifically for property and property value labels. The scores represent the percentage of labels available in a given language.
  • ShExStatements ShExStatements allows the users to generate shape expressions from simple CSV statements, CSV files and Spreadsheet. It can be used from the command line as well as from the web interface.
  • WDProp WDProp provides a visual interface to understanding and improving multilingual and collaborative ontology development on Wikidata. Its goal is to understand different aspects of Wikidata properties, particularly focusing on the translation process.
  • WikiProvenance WikiProvenance provides the following features: statistics on statements with references, use of external identifiers, count and details of multilingual articles on Wikipedia, Wikinews, Wikibooks, Wikisource, Wiktionary, Wikiquote, Wikivoyage, Wikiversity, Wikispecies.
  • WBProp WBProp aims to provide every possible information on properties on Wikibase and Wikidata.
