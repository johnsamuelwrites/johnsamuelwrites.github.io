There are several tools and applications out there, a significant number of them open-source, yet to be explored, analysed and verified whether they meet our various requirements. There are some requirements that we even don't know until we heard for the very first time about a particular tool or an application. Sometimes our keyword search terms do not return the tool that we are looking for. The time spent searching, analysing these tools and applications and finally deciding whether to continue with our choice is sometimes very high.

What do tools and applications serve? Avoid Repitition and automate routine tasks

I don't think I can call any tool as the 'right' tool. It's subjective. yet there can be several parameters to make comparison: user interface, effort (length of commands, options, any aliases allowed) required to achieve a task, learning curve, number of features (but saying more features means better tool may not be correct). I have quite often seen others using certain applications with so much ease that I myself have struggled to use and vice versa. Therefore it's really hard to judge an application.
