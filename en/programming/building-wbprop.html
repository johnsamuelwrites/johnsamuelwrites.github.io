Setup

First of all, create the directory wbprop. 

	  
$ mkdir wbprop
$ cd wbprop

Initialize the package.json using the following command npm init. It will auto-generate the file as shown below. 

	  
$ npm init
This utility will walk you through creating a package.json file.
It only covers the most common items, and tries to guess sensible defaults.

See `npm help json` for definitive documentation on these fields
and exactly what they do.

Use `npm install ` afterwards to install a package and
save it as a dependency in the package.json file.

Press ^C at any time to quit.

$ npm install --save-dev webpack webpack-cli webpack-dev-server

Next, index.html is written as below

	  
<html>
    <head>
        <title>WBProp: Everything about Wikibase Properties</title>
    </head>
    <body>
        <p>WBProp: Everything about Wikibase Properties</p>
        <script src="wbprop.js"></script>
    </body>
</html>

Next, wbprop.js is written as below

	  
console.log("WBProp: Hello!");

In order to specify the entry, webpack.config.js is written as below

	  
module.exports = {
  entry: './wbprop.js'
};

Now run the following code in order to run the basic application

	  
$ npm test
> wbprop@0.1.0 test /home/john/contributions/wbprop
> webpack-dev-server

Now open the following link http://localhost:8080/ on the browser. If everything is fine, you can see WBProp both on the web page as well as the console.
