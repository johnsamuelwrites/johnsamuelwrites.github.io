Large language models based search engine optimization

John Samuel

NOTE: Article in Progress

The web is constantlu evolving and so does user behaviour From consulting the websites for the latest news to checking the news on the social media sites, the change was very rapid With the constant development and enhancements of the search engines, there is no need to consult websites for a number of simple questions like getting the local temperature, the height of a mountain, the distance between two cities, the population of a country, etc. There is no more need to use keywords to make these searches. It is possible to converse directly to the search engines.

When the large language models1 are connected directly to the internet Integration of large language models like chatgpt with the search engines.

From article excerpts to information boxes on the search engines, the web has changed very rapidly. It is slowly resulting in a behaviour where the user no longer needs to click the outgoing links in the search results, but can obtain the answer from the main site. Everything in one single place, from questions to answers without leaving the site. This big vision is slowly taking hold as the default behavior.
